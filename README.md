# AI in Healthcare

FastAPI app deploying a fine-tuned VGG16 model for Knee Osteoarthritis Classification from X-ray images, exposing a simple API endpoint for image uploads and real-time predictions.

## Quick Start

Run the following command to build images from Dockerfiles and start all services:

```bash
cd app
docker compose up --build
```
Stop the services
```bash
docker compose down
```
